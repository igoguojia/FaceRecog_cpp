# FaceRecog_cpp
FaceRecog
## 一、	实验目的：
通过建立自己的人脸检测识别系统，深入理解人机交互设计。
## 二、	实验要求
能够调用摄像头实时抓取图像，如果图像中出现人脸，能够检测定位人脸，同时保存人脸，建立自己的人脸样本集；训练人脸识别模型；测试人脸识别模型。
## 三、	实验结果分析
### 一键录入人脸数据，自动打标签，并且训练模型。
### 能够有效检测人脸数据
### 对于陌生人进行预警
### 记录并保存检测者照片
